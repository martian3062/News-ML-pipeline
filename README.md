# **NewsAI — AI-Native Business Intelligence**

> **Frontend:** [https://news-llm.netlify.app/](https://news-llm.netlify.app/)  
> **Backend:** [https://ai-news-fvca.onrender.com/](https://ai-news-fvca.onrender.com/)

---

## 🌟 Top Features

*   🎥 **AI Video Studio**: Transform any article slug into a full-length AI-narrated video broadcast. Supports native seeking/scrubbing via HTTP Range streaming.
*   💬 **AI Concierge**: An intelligent agentic chat interface for deep-diving into news insights.
*   🗞️ **Intelligent News Synthesis**: Multi-layer pipeline for real-time scraping, classification, and summarization of global news.
*   🎙️ **Multi-lingual Voiceover**: Integrated Edge TTS with Llama-3.3-70B for high-fidelity news reporting.
*   ✨ **Glassmorphic UI**: Premium aesthetics using Framer Motion, GSAP, and Tailwind CSS for a state-of-the- art visual experience.

## 🛠️ Tech Stack

### **Frontend**
- **Next.js 14** (App Router)
- **Framer Motion** & **GSAP** (Smooth Animations)
- **Lucide Icons** & **Tailwind CSS** (Styling)

### **Backend**
- **Django 5.1** & **Django REST Framework**
- **Gunicorn** & **WhiteNoise** (Production Serving)
- **dj-database-url** (PostgreSQL/Supabase Integration)

### **AI Pipeline**
- **Groq (Llama-3.3-70B)**: Scripting & Intelligence
- **Google Gemini**: Large Context Analysis
- **Edge TTS**: Audio Synthesis
- **Pexels API**: Dynamic Stock Footage
- **Llama-4-Scout**: Vision Frame Validation
